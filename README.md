# scripts

My personal assortment of utility scripts.

## Files

- `ubuntu-dev-setup.sh`

  Intended for a fresh `Ubuntu 16.04` installation, this script will install the following:

  - `git`
  - `php7.2` + extensions
  - `composer`
  - `mysql`
  - `laravel`
  - `python3.6`
  - `pip`
  - `virtualenvwrapper`
  - `zsh`
  - `oh-my-zsh`
